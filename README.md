## About

I design and build backend systems, mobile apps, and automation tools at **[Tafi Solutions](https://tafisolutions.com)** — a boutique software studio from Moldova. Most of what we ship lives in private repos, but here's what I work on:

[**Fleet management**](https://gps.tafi-agent.com) — real-time GPS tracking, route optimization, driver analytics<br/>
[**Transport management System**](https://tafi-agent.com) — The all-in-one platform to run your transportation business<br/>
**Automation & bots** — Telegram bots for freight monitoring, loyalty systems, workflow tools<br/>
**Mobile apps** — driver management, booking systems, field service tools (Flutter)<br/>
**Web platforms** — freight exchanges, corporate sites for operators across 18+ European countries
